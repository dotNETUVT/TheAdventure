enum TheAdventure.KeyCode : UInt16

Represents key codes for keyboard input.

This enumeration covers a broad range of keyboard keys and is used to map physical key states to actions within the game.

◆ MouseButton

enum TheAdventure.MouseButton : byte

Defines identifiers for mouse buttons recognized in the game.

This enumeration lists all mouse buttons that can be pressed and detected within the game context, including standard buttons like primary (left click) and secondary (right click), as well as additional buttons for mice that support more inputs.

Enumerator
Primary 

The primary mouse button, usually the left button.

Middle 

The middle mouse button, often integrated with the scroll wheel.

Secondary 

The secondary mouse button, usually the right button.

Count 

Represents the count of mouse buttons recognized by the game.
