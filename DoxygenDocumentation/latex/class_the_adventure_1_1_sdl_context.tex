\doxysubsection{Detailed Description}
Manages the loading and access of native SDL library functions. 

{\ttfamily Sdl\+Context} is responsible for loading the SDL library dynamically at runtime and providing access to its functions. It supports multiple platforms by loading the corresponding native library according to the running OS and architecture. This class implements {\ttfamily INative\+Context} to be used with Silk.\+NET for direct SDL interop calls.
